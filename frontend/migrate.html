🔄 Migrar Cotizaciones al Servidor MySQL

Esta herramienta transfiere las cotizaciones guardadas localmente en este navegador hacia la base de datos MySQL centralizada.

Paso 1: Abre esta página en Google Chrome (donde tienes las cotizaciones)
Paso 2: Haz clic en "Migrar Ahora"
Paso 3: Abre la misma URL en Edge y recarga el Historial — aparecerán todas
